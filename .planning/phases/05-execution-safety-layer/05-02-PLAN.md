   d. Otherwise -> use `from langgraph.types import interrupt` (LangGraph's interrupt mechanism). Call `interrupt({"type": "hitl_review", "generated_sql": sql, "sql_explanation": explanation, "relevant_tables": tables})`. This pauses the graph. When resumed, the runtime re-enters the node with updated state containing approval_status.
